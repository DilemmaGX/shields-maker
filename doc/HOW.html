How to use it

Introduction

This repo (shileds-maker) is a project to help non-programmers and programmers to generate basic static badge easily. The badge generator is based on , an enhanced version of shields.io badges, supporting Octions and Feather icons.

Page designed with Bootstrap v5.3.2.

User Interface

The user interface is basically composited of two parts: Generator and Quick Reference.

The Generator part basically handles the main generation functions to generate a badge. For This version, the Generate supports URL, Markdown, rSt, AsciiDoc, HTML, and BBCode.

The Quick Reference (QR) contains five iframe elements in current version. The QR contains a doc tab (this document), Shields.io Cite, and three icon libraries, supported by the Generator.

Generator

To generate a badge, a Right text is required. Other components are optional, for more details, see Shields.io Cite tab.

Note

In the original badge core, some characters are transcript differently in the url, but the core of this project is able to convert these characters, so you don't have to worry about this.
